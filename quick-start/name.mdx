---
title: "Быстрый старт"
---

Gramax — это **бесплатное** приложение с открытым исходным кодом для создания, редактирования и публикации документации. Gramax чаще всего используется как:

* Портал документации о продукте.

* Внутренняя база знаний.

* Проектная документация.

* Персональные заметки.

## Начало работы[](https://gram.ax/resources/docs/quick-start#%D0%BD%D0%B0%D1%87%D0%B0%D0%BB%D0%BE-%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D1%8B)

1. [Создайте каталог](https://gram.ax/resources/docs/quick-start/create-catalog).

2. [Опубликуйте каталог в хранилище](https://gram.ax/resources/docs/quick-start/publish-repository).

3. [Разверните портал для читателей](https://gram.ax/resources/docs/quick-start/own-server).

4. [Загрузите каталог на созданный портал](https://gram.ax/resources/docs/quick-start/add-catalog-to-docportal).

## Компоненты[](https://gram.ax/resources/docs/quick-start#%D0%BA%D0%BE%D0%BC%D0%BF%D0%BE%D0%BD%D0%B5%D0%BD%D1%82%D1%8B)

1. [Приложение Gramax](https://gram.ax/resources/docs/app). Позволяет пользователям создавать [каталоги](https://gram.ax/resources/docs/catalog) и редактировать [статьи](https://gram.ax/resources/docs/article). Есть браузерная и десктопная версия для Windows, Mac, Linux.

2. [Хранилище](https://gram.ax/resources/docs/storage). С его помощью происходит [синхронизация](https://gram.ax/resources/docs/collaboration) каталогов между пользователями. А также из хранилища информация публикуется на [портал документации](https://gram.ax/resources/docs/doc-portal).

3. [Портал документации](https://gram.ax/resources/docs/doc-portal). Это сайт, на котором читатели могут просматривать опубликованные материалы.
