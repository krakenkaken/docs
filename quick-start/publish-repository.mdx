---
title: "Опубликовать в хранилище"
---

1. [Опубликуйте каталог и статьи](https://gram.ax/resources/docs/collaboration/publish), нажав значок с облаком  в синей панели.

2. Добавьте новое [хранилище](https://gram.ax/resources/docs/storage) [GitHub](https://gram.ax/resources/docs/en/storage/github).

3. Следуйте подсказкам интерфейсна для аутентификации.

4. Подключите хранилище — репозиторий с созданным каталогом будет автоматически создан в указанном хранилище.
